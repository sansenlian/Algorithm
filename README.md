- 图论：最短路径、最小生成树、网络流建模
- 动态规划：背包问题、最长子序列

数据结构：

- 数组与链表：单 / 双向链表
- 栈与队列
- 哈希表
- 堆：最大堆 ／ 最小堆
- 树与图：最近公共祖先、并查集
- 字符串：前缀树（字典树） ／ 后缀树
